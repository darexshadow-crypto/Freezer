# ❄ FREEZER v1.0

Утилита для временной заморозки Windows-процессов.

---

## Установка и запуск

1. Убедись, что установлен **Python 3.8+** (https://python.org)
2. Зависимости ставятся автоматически при первом запуске, но можно вручную:

```
pip install psutil pynput
```

3. Запустить:

```
python freezer.py
```

> Для заморозки системных/защищённых процессов — запускай **от имени администратора**.

---

## Использование

| Поле | Описание |
|---|---|
| **Имя процесса** | Например: `notepad.exe` или просто `notepad` |
| **Длительность (мс)** | На сколько миллисекунд заморозить. Кнопки 100/250/500/1000 — быстрые пресеты |
| **Хоткей** | Нажми «НАЗНАЧИТЬ», затем любую клавишу или боковую кнопку мыши |
| **❄ ЗАМОРОЗИТЬ** | Заморозить вручную |
| **💾 СОХРАНИТЬ** | Сохранить настройки в `freezer.json` (рядом с файлом) |

### Поддерживаемые хоткеи
- Любые клавиши клавиатуры: F1–F12, Delete, Insert, Home, цифры, буквы...
- Боковые кнопки мыши: **MOUSE4** (назад) и **MOUSE5** (вперёд)
- ESC во время назначения — отменить

### Настройки
Сохраняются автоматически при закрытии окна в файл `freezer.json`.

---

## Как работает

Использует Windows API `NtSuspendProcess` / `NtResumeProcess` через `ntdll.dll` —
те же вызовы, что использует Process Hacker и аналоги.
